# 1. معلومات المشروع

## اسم المشروع بالعربية
تطبيق موبايل لإدارة الحركة والنقل في جامعة إقليم سبأ

## اسم المشروع بالإنجليزية
University Transportation Management Mobile Application

## وصف المشروع
تطبيق موبايل ذكي يهدف إلى إدارة وتنظيم عمليات النقل الجامعي داخل جامعة إقليم سبأ، حيث يوفر منصة رقمية تساعد على متابعة وإدارة بيانات الطلاب، الحافلات، السائقين، المسارات، والرحلات.

يسمح التطبيق للمستخدمين بالوصول إلى خدمات النقل الجامعي بسهولة، مثل متابعة الرحلات، تسجيل الحضور باستخدام QR Code، استقبال الإشعارات، والتواصل مع إدارة الحركة والنقل.

يعتمد التطبيق على Backend API لإدارة البيانات وربط تطبيق الموبايل بقاعدة البيانات.

# 2. نبذة عن المشروع

تطبيق موبايل إدارة الحركة والنقل في جامعة إقليم سبأ هو تطبيق ذكي يهدف إلى رقمنة عمليات النقل الجامعي وتحسين طريقة إدارة الحافلات والطلاب والرحلات.

يساعد التطبيق إدارة الحركة والنقل على تنظيم البيانات ومتابعة العمليات اليومية بشكل أسرع وأكثر دقة، كما يوفر للطلاب إمكانية متابعة خدمات النقل من خلال الهاتف المحمول.

يتكون المشروع من جزأين رئيسيين:

- تطبيق موبايل Mobile Application للمستخدمين.
- Backend API يعمل كخادم لمعالجة البيانات وربط التطبيق بقاعدة البيانات.

---

# 3. المشكلة التي يعالجها المشروع

تعتمد إدارة النقل الجامعي في بعض الأحيان على الطرق التقليدية، مما يؤدي إلى:

- صعوبة إدارة بيانات الطلاب والحافلات.
- الاعتماد على الملفات والجداول اليدوية.
- صعوبة متابعة الرحلات اليومية.
- صعوبة تسجيل حضور الطلاب.
- تأخر تحديث معلومات النقل.
- ضعف التواصل بين إدارة النقل والطلاب.
- صعوبة استخراج التقارير.

---

# 4. أهداف المشروع

يهدف تطبيق الموبايل إلى:

- تطوير تطبيق ذكي لإدارة الحركة والنقل الجامعي.
- تنظيم بيانات الطلاب والحافلات والسائقين.
- تسهيل متابعة الرحلات والمسارات.
- توفير تسجيل حضور الطلاب باستخدام QR Code.
- تحسين التواصل بين المستخدمين وإدارة النقل.
- إرسال الإشعارات المهمة للمستخدمين.
- توفير تقارير تساعد الإدارة في اتخاذ القرارات.
