2. نبذة عن المشروع

يهدف تطبيق إدارة الحركة والنقل في جامعة إقليم سبأ إلى تحويل إدارة النقل الجامعي من الإجراءات التقليدية واليدوية إلى نظام إلكتروني متكامل يسهل متابعة الحافلات والطلاب والسائقين والرحلات بشكل لحظي.

يوفر النظام أدوات لإدارة بيانات الطلاب والسائقين والحافلات والمسارات والرحلات والاشتراكات والإشعارات والتقارير، بالإضافة إلى تطبيق موبايل يتيح للطلاب متابعة رحلاتهم وتسجيل الحضور  واستلام التنبيهات.

يتكون المشروع من جزأين رئيسيين:

تطبيق موبايل للطلاب.

نظام خلفي (Backend API) لإدارة البيانات.
